# Varun M - Portfolio

A modern, Apple-inspired personal portfolio website showcasing skills, projects, experience, and contact information.

## Features

- Dark theme with glassmorphism effects
- Responsive design for all devices
- Smooth scroll and subtle animations
- Contact form (ready to integrate with Formspree or EmailJS)
- Sections: Home, About, Skills, Projects, Experience, Certifications, Contact

## Technologies Used

- HTML5
- CSS3 (with CSS variables, flexbox, grid)
- Vanilla JavaScript
- Google Fonts (Poppins)

## Deployment

This site is ready to be deployed on GitHub Pages, Vercel, or any static hosting service.

## Contact Form

To enable the contact form, replace the placeholder in `assets/js/script.js` with your Formspree or EmailJS endpoint.
